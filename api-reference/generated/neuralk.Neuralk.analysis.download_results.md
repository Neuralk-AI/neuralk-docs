# Neuralk.analysis.download_results

#### Neuralk.analysis.download_results(analysis, folder_path)

Save the results of an analysis to a local folder using its ID or object.

* **Parameters:**
  * **analysis** ([*str*](https://docs.python.org/3/library/stdtypes.html#str) *|* *Analysis*) – The analysis object or its id.
  * **folder_path** ([*str*](https://docs.python.org/3/library/stdtypes.html#str)) – The folder where to save the result files.
* **Raises:**
  **NeuralkException** – If the result download fails.
