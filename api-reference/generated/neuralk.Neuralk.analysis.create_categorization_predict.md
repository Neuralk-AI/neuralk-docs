# Neuralk.analysis.create_categorization_predict

#### Neuralk.analysis.create_categorization_predict(dataset, name, categorization_fit_analysis)

Create a new categorization predict analysis from a fit analysis hash ID.

* **Parameters:**
  * **dataset** (*Dataset*) – The dataset to use for prediction.
  * **name** ([*str*](https://docs.python.org/3/library/stdtypes.html#str)) – The name of the prediction analysis.
  * **categorization_fit_analysis** ([*str*](https://docs.python.org/3/library/stdtypes.html#str) *|**Analysis*) – The object or ID of the fit analysis.
* **Returns:**
  The created prediction analysis object.
* **Return type:**
  Analysis
* **Raises:**
  **NeuralkException** – If the analysis creation fails.
