# Neuralk.projects.create

#### Neuralk.projects.create(name, exist_ok=False)

Create a new project.

* **Parameters:**
  * **name** ([*str*](https://docs.python.org/3/library/stdtypes.html#str)) – The name of the project to create
  * **exist_ok** ([*bool*](https://docs.python.org/3/library/functions.html#bool)) – If True, return the project without errors if it exists.
* **Returns:**
  The created or found project object
* **Return type:**
  Project
* **Raises:**
  **NeuralkException** – If the project creation fails
